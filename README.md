### 📖 Overview

A real-time Vehicle Health Monitoring ECU built using **STM32 NUCLEO-G071RB** and **NXP i.MX 8M Plus EVK**. The firmware acquires live vehicle parameters and sends them to an Embedded Linux system for monitoring and diagnostics.

- 🚗 Vehicle Speed Monitoring
- ⚙ Engine RPM Monitoring
- ⛽ Fuel Level Monitoring
- 🌡 Engine Temperature Monitoring
- 📡 UART Communication
- 🐧 Embedded Linux Dashboard
- 🔧 Hardware–Software Integration
- 🧪 Firmware Validation & Testing

**Tech Stack:** STM32 • Embedded C • UART • ADC • NXP i.MX 8M Plus • Embedded Linux
